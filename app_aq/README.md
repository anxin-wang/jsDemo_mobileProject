1 文件夹说明
/images 所有图片的文件夹 该文件夹下对应每个模块的名称文件夹
/style 所有样式文件夹 
/javascript 所有脚本文件夹

2 命名规范
用户中心模块 - u_为前缀  + m_为动作(增加 a_ / 删除 d_ / 查询s_ / 修改 m_)(可选) + nick为文件名
答疑模块 - aq_为前缀  + m_为动作(增加 a_ / 删除 d_ / 查询s_ / 修改 m_)(可选) + nick为文件名      

2 模块划分
~ user - 用户中心模块文件夹
~ aq - 答疑模块文件夹
~ help - 帮助页面
